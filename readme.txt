== Installation ==

1. In your WordPress dashboard, go to **Plugins > Add New**
2. Search for **Scoby Analytics** and click **Install Now**
3. Click **Activate**
4. Navigate to **Scoby Analytics** in the sidebar
5. Follow the guided setup steps to configure your workspace
